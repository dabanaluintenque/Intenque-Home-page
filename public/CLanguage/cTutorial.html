Sorting Algorithms

Selection sort

Sudocode:
For i from 0 to n - 1
Find smalesst number between numbers[ i ] and numbers[ n - 1 ]
Swap smallest number with numbers[ i ]

formula: n(n-1)/2
running time: O(n^2)

bobble sort

sudocode:
Repeat n-1 times
For i from 0 to n - 2
If numbers [ i ] and numbers [ i + 1 ] out of order
Swap them
If not swap
quit

Formula: n^2 - 2n + 1
running time: O(n^2)


Binary Search

sudocode:
If no doors left
Return false
If number behind middle door
Return true
Else if number < middle door
Search left half
Else if number > middle door
Search right half
running time: O(log n)

merge sort

sudocode:
If only one nmber
Quit
Sort left half of numbers
Sort right half of numbers
Merge sorted halves

running time: O(n log n)
