How to add a new user in Linux using terminal?

step 1: open terminal
step 2: login as root in terminal using this command "sudo su -"
step 3: enter your root password
step 4: type "sudo adduser username" Note: change the username by the name of the user you want to add. Also make sure you start username with a lower case letter.
