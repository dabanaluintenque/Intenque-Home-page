Network Data Transmission

Copper Cable
Copper Cable is used to transmit electrical signals. The cable between two nodes creates a low voltage electrical circui betwwen the interfaces on the nodes. There are two main types of copper cabel: twisted pair and coaxial (coax). Copper cable suffers from high attenuation, meaning that the signal quickly loses strength over long links. Twisted pair cable is rated to Category (Cat) standards.

Fiber Optic Cable
Fiber Optic cable carries very high frequency radiation in the infrared light part of the electromagnetic spectrum. fiber optic cable supports higher bandwidth over longer links than copper cable. Fiber optic cabling is divided into Single Mode (SMF) and MultiMode (MMF) types, MMF is categorized by Optical Mode designations (OM1, OM2, OM3, and OM4).

Ethernet Standards / Media Access Control and Collision domains

Ethernet is a multiple access area network, which means that the available communications capacity is shared betwwen the nodes that are connected to the same media.
Ethernet uses a contention-based MAC system. Each network node connected to the same media is in the same Collision domain. When two nodes transmit at the same time, the signals are said to collide and neither signal can reach its destination.
The Ethernet protocol governing contention and media access is called Carrier Sense Multiple Access with Collision Detection (CSMA/CD).
The collision detection mechanism means that only half-duplex transmisions is possible.
In the 10BASE-T wiring topology each node is cabled to an Ethernet hub. The 10BASE-T standard dates from 1990.

100BASE-TX Fast Ethernet Standard

The Fast Ethernet standard users the The same CSMA/CD protocol as 10BASE-T.
100BASE-TX refers to Fast ethernet working over Cat 5 ( or better) twisted pair copper cable with a maximum supported link lenght of 100 meters (328 feet).
