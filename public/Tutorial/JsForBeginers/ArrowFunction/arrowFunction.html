Arrow Functions

What does a arrow function allow us to do?

Arrow function allow us to write shorter syntax like:

let createFunction = (x, y) => x + y;
console.log(createFunction(10, 3));

Arrow function with using just parentheses ()

Syntax:

let myFunction = ( ) => {
return "This is how to use arrow function"
console.log(myFunction());
}

Another way to write the function above is:
let myFunction = ( ) => "This is how to use arrow function";
console.log(myFunction());

Note:
The second syntax works because the function has only one statement, and the statement returns a value. We can remove the brackets and the return keyword:

Arrow function with parameters

When you have a parameters pass them inside parentheses

Syntax:

let myFunction = (value) => "Say something" + value;
console.log(myFunction("my friend"));

Another way the write the function above is without parentheses.

let myFunction = value => "Say something" + value;
console.log(myFunction("my friend"));
Note:
It is the fact that we have only one parameter, we skip the parentheses.
