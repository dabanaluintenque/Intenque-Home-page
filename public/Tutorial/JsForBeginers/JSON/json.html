JSON

What is JSON often used for?
JSON is often used when data is sent from a server to a web page.
What is JSON?
  • JSON stands for JavaScript Object Notation.
  • JSON is language independent *

JSON Example:

{
"Students": [
{"firstname": "Dabana", "Age":"23"},
{"firstname": "Erwin", "Age":"21"},
{"firstname": "Keanu", "Age": "20"}
]
}

JSON Syntax Rules

  • Data is in name & value pairs
  • Data is separated by commas
  • Curly braces hold objects
  • Square brackets hold arrays

Converting a JSON Text to a JavaScript Object

First, create a JavaScript string containing JSON syntax.

let values = '{"students": [' +
'{"fname":"Dabana", "Age":"23"},'+
'{"fname":"Erwin", "Age":"20"} ]';

How to convert string into JavaScript Object?

Use the JavaScript built-in function JSON.parse( ) to convert the string into javaScript object.

How to convert the object back into String?

Use the JavaScript buit-in function JSON.stringfy( ) to convert it back to string.
