Loops

Loops execute a block of code number of times.
Use loops if you want to run the some code over and over again, each time with a different value.

There are different kinds of Loops in JavaScript

  • for loops loops through a block of code a number of times.
  • for/ in loops through the properties of an object.
  • for/ of loops through the values of an iterable object.
  • while loops through a block of code while a specified condition is true.
  • do/while also loops through a block of code while a specified condition is true.

for loop

for loop loops through a block of code a number of times.
Syntax:
for( statement 1; statement 2; statement 3){
// block of code to be executed
}

statement 1 is executed one time before the execution of the code block. (Starting point).
statement 2 defines the condition for executing the code block. (How many time the loop will run?)
statement 3 is executed every time after the code block has been executed. (Increase the value each time the code block run).
Example:
Enter the six hex decimal letters to know their values. (A-F)

for in

for in loops through the properties of an Object.
Syntax:
for(let key in object){
// block of code to be execute
}

Example:
const person = {firstName: "Dabana", lastName ="Intenque", age = 23};
let values ="";
for(let i in person){
values += person[i];
}
Note:
  • The for in loop iterates over a person object
  • Each iteration returns a Key(i)
  • The key is used to access the values
  • The values of the key is person[i]
Note:
for in Over Arrays can also loop over the properties of an Array:
But.
Do not use for in over an Array if the index order is important.
The index order is implementation-dependent, and array values may not be accessed in the order you expect.
It is better to use a for loop or Array.forEach() when the order is important.

Array.forEach()

The forEach() method calls a function (a callback function) once for each array element.
Syntax:
const arrayName =[item1, item2, item3, ... , itemN];
let text ="";
arrayName.forEach(functionName);
document.getElementById('idName').innerHTML = text;
function functionName(value, index, array){
text += value +"<br>"
}
 Note:
The functionName takes 3 arguments:
  • The item value
  • The item index
  • The array itself
But we only use one parameter (value).
Which mean we can rewrite functionName(value); with only value parameter if we want.
Example:
Enter the name of your five closest friends.

for of loop

for of loop through the values of an iterable object.
It lets you loop over iterable data structures such as Arrays, Strings, Maps, NodeLists, and more:
Syntax:
for(let variable of iterable){
// block of code to be execute
}
Note: iterable can be an array, object, string ...
Example:
Name your five favority courses.

while loop

while loop loops through a block of code as long as a specified condition is true.
Syntax:
while (condition){
// block to execute the code.
}
Example:
let value ="";
let i = 0;
while(i < 7){
value += i;
i ++;
}
document.getElementById('idName').innerHTML = value;
