Regular Expressions

What is a regular Expression?

Regular Expression is a sequence of the characters that form a search pattern.
Example:
Searching for a data in a text, we can use this search pattern to describe what we are searching for.
Where to use Regular Expression
Regular Expression can be used to perform all types of text search and text replace.
Syntax:
/pattern/modifiers;
/was/i;
was is the pattern and i is the modifier
Ex.
const text = "Today was a good day."
let findText = /was/i;
console.log(findText.test(text)); Output = true;
console.log(text.search(findText)); Output = 6

Using String Methods

As we said above regular expressions are often used with the string methods: search() and replace().

String search() with Regular Expression

search() method searches a string for a specified value and returns the position of the match:
Syntax:
const text = "Today was a good day."
text.search(/good/i);

String replace() with Regular Expression

replace() method replace a given value with the existent value on a string.
Syntax:
const text = "Hard work pays off.";
text.replace(/off/i, "if you are consistent.")

Regular Expression Modifiers

What to use modifier for?
Modifiers can be used to perform case-insensitive more global searches:

Modifier Description Example
i Perform case-insensitive matching. const Q = "What is an Amendment? R: is change and addition in the contitution"
Q.search(/amendment/i);
Note: Amendment starts with upper case latter A
but in the search we used lower case letter a. It will works.
g Perform a global match (find all match rather than stopping after the first match) const Q = "What do we call the first ten amendments?
We call the first ten amendments the Bill of rights.";
Q.match(/call/g); Output call, call;
Note: RegExp did not stop after the first call.
gi Perform a global matching including the upper case letter. const Q = "What is one right or freedom from the first Amendment?
One right or freedom from the first amendment is speech";
Q.match(/amendment/gi); Output Amendment, amendment
Note: It finds both with a lower case letter and upper case letter.
m Perform multiline matching

Regular Expression Patterns

Brackets are used to find a range of characters:
Expression Description
[abc] Find any of the characters between the brackets
[0-9] Find any of the digits between the brackets.
(x|y) Find any of the alternatives separated with |

Find any of the characters between the brackets.
