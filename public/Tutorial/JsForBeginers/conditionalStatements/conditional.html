Those are the JavaScript conditional Statements: if, else, else if and switch.
  • if- is used to specify a block of code to be executed, if a specified condition is true.
  • else- is used to specify a block of code to be executed, if the same contional is false.

if Statement

if- is used to specify a block of code to be executed, if a specified condition is true.
Syntax
if (condition){
// block of code to be executed if the condition is true
}
Example: say something if study hours is greater than 5.

else Statement

We use else statement to specify a block of code to be executed if the given condition is false.
Syntax:
if(condition){
// block of code to be executed if the condition is true
}
else{
// block of code to be executed if the condition is false
}

else if statement

We use else if statement to specify a new condition if the first condition is false.
Syntax
if(condition1){
// block of code to be executed if the condition1 is true
}
else if(condition2){
// block of code to be executed if the condition1 is false and condition2 is true
}
else{
// block of code to be executed if both condition1 & 2 are false
}

Ternary Operator (?:)

The conditional ternary operator assigns a variable based on some contidaion.
Syntax:
let answer = (condition)? "Do something if condition is true": " else do something if condition is false";
Example:
let answer = (10 < 20)? "10 is less than 20" : "10 is not less than 20";

Switch Statement

We use switch statement to select one of many code blocks to be executed.
Syntax:
switch(condition){
case x:
// code block to be executed
break;
case y:
// code block to be executed
break;
case z:
// code block to be executed
break;
default:
// code block to be executed if none the statement above applies
}
How switch statement works?
  • The switch expression is evaluated once.
  • The value of the expression is compared with the values of each case.
  • If there is a match, the associated block of code is executed.
  • If there is not match, the default code block is executed.
Note:

Switch cases use strict comparision(===)
The values must be of the same type of match.
A strict comparision can only be true if the operands are the same type.
