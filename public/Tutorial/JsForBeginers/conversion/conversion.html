Conversion

How to convert javaScript variables?

JavaScript variables can be converted to a new variable and another data type:
  1. By the use of javaScript function
  2. Automatically by JavaScript itself

Converting string to Numbers

The global method to convert a strings to numbers is Number()
E.g.
let pi = "3.14";
let text = " ";
let empty = "";
let space = "77 13";
Number(pi); output = 3.14
Number(text); output = 0
Number(empty); output = 0
Number(space); output = NAN

Number Method

Method Description
Number() Returns a number, converted from its argument.
parseFloat() Parses a string and returns a floating point number.
parseInt() Parses a string and returns an interger

Converting a Number to String

The global method to convert a number to String is String()
Also, the number method toString() does the same.
E.g.
let pi = 3.14;
let number = 77;
let otherNumber = 13;
String(pi); output a string of number = 3.14
pi.toString(); output a string of number = 3.14
String(number); output a string of number = 77
number.toString(); output a string of number = 77
String(othernumber); output a string of number = 13
otherNumber.toString(); output a string of number = 13

Convertng Booleans to Number

The global method to convert booleans to a number is Number()
let x = false;
let y = true;
Number(x); Output = 0
Number(y); Output = 1
