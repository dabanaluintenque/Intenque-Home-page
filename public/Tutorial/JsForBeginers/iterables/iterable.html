Iterable

We use iterable to iterate objects. Example arrays.
Iterable can be iterated over with for of loop.
Itarable simple mean loop through a sequence of elements.

    We can use iterable to iterate over:

  • String
  • Array
  • Sets
  • Maps
  • NodeLists
    • and more:

for of loop

for of loop statement loops through the elements of an iterable object.

Syntax:
for(variable of iterable){
// code block to be executed
}
 Example
for(let i of arrayName){
values += i;
}
Iterate over a string and return the value of each letter in Ascii
ASCII stands for (American Standard Code for Information Interchange)
ASCII is a subset of Unicode and is made up of 128 symbols in the character set.
