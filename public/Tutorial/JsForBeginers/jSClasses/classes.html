The constructor Method

constructor is a special method of a class.
  • The name has to be constructor
  • it is executed automatically when a new object is created
  • it is used to initialize object properties

Class Methods

Steps:
  1. step: use the keyword class to create a class.
  2. step: Always add a constructor( ) method.
  3. step: add any number of methods you want.
Syntax:

}
