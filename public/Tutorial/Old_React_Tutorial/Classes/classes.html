What is a Class?

A class is a type of funcion, but instead of using the keyword function to initiate it, we use the keyword class, and the properties are assigned inside a constructor( ) method.

Note: for the standard naming convention is to begin each class name with an uppercase letter.

Syntax:
class Vessels{
constructor(vesselName){
this.myVesselName = vesselName;
}
}

Create an object called "myVessel" based on the Vessels class:

const myVessel = new Vessels("directToBissau");
